# SehatMitra Landing Page

Demo landing page for SehatMitra with links to all three portals.

## Setup

```bash
npm install
npm start
```

The landing page will run on http://localhost:3004

## Portals

- **Patient Portal**: http://localhost:3000
- **ASHA Worker Portal**: http://localhost:3001
- **Doctor Portal**: http://localhost:3002

Make sure all three portals are running on their respective ports before using the landing page.
